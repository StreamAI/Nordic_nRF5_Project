1. 连接DK52开发板，运行flash_hex_to_nRF52DK.bat 烧录hex程序；
2. 打开串口工具连接DK52开发板串口，波特率115200；
3. 勾选“发送新行”以字符串形式发送命令“RST\n”开发板重置；
4. 发送命令“CONN:MAC”（比如“CONN:E10AEBA5FBF8\n”）开发板开始尝试连接目标MAC蓝牙设备，需要确保目标设备处于广播状态可以被扫描到，待连接成功后串口工具会返回“Connecting to target device:MAC.“字符串；
5. 发送命令“RSSI\n”返回当前连接的信号强度（比如”RSSI: -36 dBm“）；
6. 发送目标MAC蓝牙设备支持的APP命令可正常获得返回值或响应（命令需以‘\r’或‘\n’结束）；
